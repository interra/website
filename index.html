Interra Open Data Catalog Tools

The goal of Interra is to provide open source and open data tools.

Libraries

The current libraries are currently being developed:

  • Data Catalog Components: React components for open data catalogs
  • Data Catalog Frontend: A React-based front-end for open data catalogs. Currently this is being utilized as the front-end for DKAN2
  • Data Catalog Admin: An adminstrative interface for open data catalogs

The following libraries are no longer under active development:

  • Catalog Generate: Node.js library for harvesting, storing, and editing open data datasets
  • dashql: dashql backend for data visualization apps
  • 311 Demo: 311 Data Visualizations for Philly and Beyond

About

Interra was started by Aaron Couch. Contributors have included Paul Walker, Janette Day, Gerardo Gonzalez and others.
